Custom Wordclocks

Goal

Make a custom wordclocks with my family member's names on them.

Overview

This year for Christmas I made these custom wordclocks for my family members. It was journey and a half. I layed a custom cut vinyl sticker over class inside an Ikea Ribba frame. I then designed and 3D printed a baffling to isolate each LED. I made an array of individual addressable RGB leds and placed them in the baffles.

The whole thing was driven using an Arduino and a real time clock module. I programmed the Arduino to display different times based on the time of the day. The code will be available on Github soon. I learned so much during this project and got a lot of interest for custom ones, so I decided to run with this a little bit. I designed an LED controller that I'm getting manufactured right now.
